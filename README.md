# Gestión de Cargos — JPA/Hibernate (Java SE)

CRUD mínimo de **Cargo** usando **Jakarta Persistence (JPA)** con **Hibernate ORM 6** y **Hibernate Validator 9** en **Java SE**.

---

## Requisitos
- **Java 17+**
- **Maven 3.8+** 
- Base de datos (se usó.: **PostgreSQL**)
